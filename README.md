# Task Manager 📝

A modern, responsive Task Management web application built to demonstrate full-stack development skills. This project showcases a seamless integration between a powerful backend and a reactive frontend without building a separate API.

## 🚀 Tech Stack

- **Backend:** [Laravel](https://laravel.com/) (PHP)
- **Frontend:** [Vue.js 3](https://vuejs.org/) (Composition API)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Integration:** [Inertia.js](https://inertiajs.com/) (Using Laravel Breeze)
- **Database:** SQLite

## ✨ Features

- **User Authentication:** Secure login and registration system provided by Laravel Breeze.
- **CRUD Operations:** Users can Create, Read, Update, and Delete their tasks.
- **Single Page Application (SPA) Experience:** Fast and smooth page transitions using Inertia.js without reloading the browser.
- **Responsive Design:** Mobile-friendly UI built completely with Tailwind CSS utility classes.


## 🎯 Learning Objectives

This project was built as a hands-on learning experience to master:
- Setting up a Laravel project with a Vue.js frontend.
- Utilizing Tailwind CSS for rapid UI development.
- Understanding the Inertia.js workflow (Passing data from Laravel Controllers directly to Vue Components).
- Managing database migrations and Eloquent ORM.
